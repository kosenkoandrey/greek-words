# Greek Words

Еженедельные выпуски греческих слов, генерируются скиллом `greek-words` из личного словаря.

Оглавление всех выпусков (новые сверху): https://kosenkoandrey.github.io/greek-words/

Каждый выпуск живёт на своём адресе `выпуски/<YYYY-Wnn>.html` и никогда не затирается.
Корневой `index.html` — автоматически сгенерированное оглавление, публикуется скриптом
`~/.claude/skills/greek-words/scripts/publish.py`.
